# Vim

* Help Navigation
    - Follow Link   Ctrl-]
    - Back          Ctrl-t

* Add Word to Dictionary
    - zg
 
* Sort
    - :sort         "normal"
    - :sort!        "reverse"

* Coc List diagnostic list
    - :CocList diagnostic

* Open Quickfix window
    - :copen
    - :cope
    - <leader> qf
 
* Show Documentation
    - K {in normal mode}

* Folding
    - Close => zc
    - Open => zo
    - Toggle => za
    - Open All => [zR](zR)
    - Close All => zM
